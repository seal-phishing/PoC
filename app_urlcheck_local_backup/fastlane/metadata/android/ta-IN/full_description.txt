URL இணைப்புகளைத் திறக்கும்போது இந்தப் பயன்பாடு ஒரு இடைத்தரகராகச் செயல்படுகிறது. நீங்கள் அவ்வாறு செய்யும்போது, URL பற்றிய தகவல்களைக் கொண்ட ஒரு சாளரம் காட்டப்படும், இது அதில் மாற்றங்களைச் செய்ய உங்களை அனுமதிக்கிறது.
மின்னஞ்சல், சமூக வலைப்பின்னல் பயன்பாடு அல்லது பிறவற்றிலிருந்து வெளிப்புற இணைப்பைத் திறக்க வேண்டியிருக்கும்போது குறிப்பாகப் பயனுள்ளதாக இருக்கும்.


URLCheck என்பது TrianguloY ஆல் உருவாக்கப்பட்டது, முதலில் தனிப்பட்ட பயன்பாட்டிற்காக. இது திறந்த மூலமாகும் (CC BY 4.0 உரிமத்தின் கீழ்), இலவசம், எந்த விளம்பரங்கள் அல்லது டிராக்கர்கள் இல்லாமல், லேசான அளவு மற்றும் தேவையான சில அனுமதிகளைப் பயன்படுத்துகிறது (இணைய அனுமதி மட்டும், பயனர் கோரும்போது மட்டுமே செய்யப்படும் தொகுதி சோதனைகளுக்கு). மாற்றங்களைப் பரிந்துரைக்க, மாற்றங்களைச் செய்ய அல்லது புதிய மொழிபெயர்ப்பை முன்மொழிய விரும்பினால், மூலக் குறியீடு GitHub இல் கிடைக்கிறது: https://github.com/TrianguloY/URLCheck


பயன்பாடு ஒரு மட்டு அமைப்புடன் கட்டமைக்கப்பட்டுள்ளது, நீங்கள் தனிப்பட்ட தொகுதிகளை இயக்கலாம்/முடக்கலாம் மற்றும் மறுவரிசைப்படுத்தலாம், அவற்றில் பின்வருவன அடங்கும்:

* உள்ளீட்டு உரை: கைமுறையாகத் திருத்தக்கூடிய தற்போதைய URL ஐக் காட்டுகிறது. முடக்க முடியாது.

* வரலாறு: பயனரிடமிருந்து வரும் திருத்தங்கள் உட்பட, பிற தொகுதிகளிலிருந்து எந்த மாற்றத்தையும் பார்த்து மாற்றியமைக்கவும் (வழக்கமான செயல்தவிர்/மீண்டும் செய் அம்சம்)

* பதிவு: சரிபார்க்கப்பட்ட அனைத்து URLகளின் பதிவையும் வைத்திருக்கிறது, அதை நீங்கள் பார்க்கலாம், திருத்தலாம், நகலெடுக்கலாம், அழிக்கலாம்...

* நிலை குறியீடு: பொத்தானை அழுத்துவதன் மூலம் அந்த URL ஐ மீட்டெடுக்க ஒரு நெட்வொர்க் கோரிக்கை செய்யப்படும், மேலும் நிலை குறியீடு காட்டப்படும் (சரி, சர்வர் பிழை, கிடைக்கவில்லை...). கூடுதலாக, அது ஒரு திசைதிருப்பலுடன் ஒத்திருந்தால், புதிய URL ஐச் சரிபார்க்க செய்தியை அழுத்தவும். URL பெறப்பட்டது, ஆனால் மதிப்பீடு செய்யப்படவில்லை, எனவே javascript ஐ அடிப்படையாகக் கொண்ட திசைதிருப்பல் கண்டறியப்படாது.

* URL ஸ்கேனர்: VirusTotal ஐப் பயன்படுத்தி URL ஐ ஸ்கேன் செய்து அறிக்கையைச் சரிபார்க்க உங்களை அனுமதிக்கிறது. அது வேலை செய்ய ஒரு இலவச தனிப்பட்ட VirusTotal API விசை தேவை. VirusTotal™ என்பது Google, Inc. இன் வர்த்தக முத்திரை.

* Url Cleaner: URL இலிருந்து பரிந்துரை மற்றும் பயனற்ற அளவுருக்களை அகற்ற ClearURLs பட்டியலைப் பயன்படுத்துகிறது. இது பொதுவான ஆஃப்லைன் URL திசைதிருப்பல்களையும் அனுமதிக்கிறது. https://docs.clearurls.xyz/latest/specs/rules/ இலிருந்து உள்ளமைக்கப்பட்ட பட்டியல்

* சுருக்கப்படாதது: URL களை தொலைவிலிருந்து சுருக்குவதற்கு https://unshorten.me/ ஐப் பயன்படுத்துகிறது.

* வினவல் நீக்கி: டிகோட் செய்யப்பட்ட தனிப்பட்ட URL வினவல்களைக் காட்டுகிறது, அதை நீங்கள் அகற்றலாம் அல்லது சரிபார்க்கலாம்.

* வடிவ தொகுதி: மாற்றுகளை எச்சரிக்கும், பரிந்துரைக்கும் அல்லது பயன்படுத்தும் regex வடிவங்களுடன் URL ஐச் சரிபார்க்கிறது. நீங்கள் உங்கள் சொந்த வடிவங்களை மாற்றலாம் அல்லது உருவாக்கலாம் அல்லது பயனர் உருவாக்கியவற்றைப் பயன்படுத்தலாம்.
உள்ளமைக்கப்பட்ட வடிவங்களில் பின்வருவன அடங்கும்:
- கிரேக்க எழுத்துக்கள் போன்ற ascii அல்லாத எழுத்துக்கள் இருக்கும்போது எச்சரிக்கை. இதைப் ஃபிஷிங்கிற்குப் பயன்படுத்தலாம்: googĺe.com vs google.com
- 'http' ஐ 'https' உடன் மாற்றப் பரிந்துரைக்கவும்
- Youtube, Reddit அல்லது Twitter ஐ தனியுரிமைக்கு ஏற்ற மாற்றுகளுடன் மாற்றப் பரிந்துரைக்கவும் [இயல்புநிலையாக முடக்கப்பட்டுள்ளது]

* ஹோஸ்ட்கள் சரிபார்ப்பு: இந்தத் தொகுதி ஹோஸ்ட்களை லேபிள் செய்கிறது, அவற்றைக் கைமுறையாகக் குறிப்பிடுவதன் மூலமோ அல்லது தொலை ஹோஸ்ட்கள் போன்ற கோப்பைப் பயன்படுத்துவதன் மூலமோ கட்டமைக்கப்படுகிறது. ஆபத்தான அல்லது சிறப்பு தளங்களைப் பற்றி எச்சரிக்க இதைப் பயன்படுத்தலாம். உள்ளமைக்கப்பட்ட உள்ளமைவு https://github.com/StevenBlack/hosts இலிருந்து StevenBlack இன் ஹோஸ்ட்களை (adware/malware, fakenews, gambling மற்றும் வயது வந்தோர் உள்ளடக்கம்) குறிப்பிடுகிறது.

* பிழைத்திருத்த தொகுதி: intent uri மற்றும் விருப்பமாக ctabs (custom tabs) சேவைபற்றிய தகவலைக் காட்டுகிறது. இது டெவலப்பர்களுக்காக வடிவமைக்கப்பட்டுள்ளது.

* திறந்த தொகுதி: திறந்த மற்றும் பகிர் பொத்தான்களைக் கொண்டுள்ளது. பல பயன்பாடுகளுடன் ஒரு இணைப்பைத் திறக்க முடிந்தால், நீங்கள் தேர்வுசெய்ய அனுமதிக்கும் ஒரு அம்புக்குறி காட்டப்படும். முடக்க முடியாது.
